Comment ajouter une serrure?


Pour un nouveau verrou, touchez pour allumer l'écran de verrouillage afin de pouvoir ajouter le verrou, puis ajoutez-le sur l'application.
Pour un verrou déjà ajouté, si vous souhaitez le lire, vous devez d'abord que l'administrateur d'origine le supprime dans l'application, puis l'ajoute de la même manière que si vous en ajoutiez un nouveau. Gardez votre téléphone connecté au réseau tout en ajoutant un verrou.

Si le verrou ne peut pas être ajouté, les raisons possibles sont les suivantes:
1. La serrure n'est pas près du téléphone
    Dans ce cas, APP ne peut pas rechercher et trouver le verrou. Lors de l’ajout de la serrure, veillez à respecter une distance de 1 mètre entre le téléphone et la serrure.
2. La serrure a déjà été ajoutée par d'autres
    Dans ce cas, l’APP peut rechercher et trouver le verrou, mais celui-ci n’est pas en mode modifiable, et le signe + n’apparaîtra pas après le nom du verrou sur l’APP. S'il vous plaît ajouter ce verrou après que l'administrateur d'origine APP le supprime.
3. La méthode d'ajout est incorrecte
    Pour ajouter un verrou, vous devez toucher et allumer l'écran de verrouillage pour que le verrou devienne un état pouvant être ajouté. Lorsque le signe + apparaît après le nom du verrou sur APP, cliquez sur le signe + pour ajouter le verrou.
