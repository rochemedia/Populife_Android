Pourquoi le mot de passe ne peut-il pas ouvrir la serrure?


Le mot de passe ne peut pas ouvrir la serrure. Les raisons possibles sont:
  1. Le mot de passe a été supprimé
  2. Le mot de passe n'a pas été utilisé une fois dans les 24 heures suivant son entrée en vigueur.
  3. Le mot de passe a expiré
  4. Le mot de passe n'est pas encore valide
  5. Le temps de verrouillage est lent, car le mot de passe n'est pas encore valide.
